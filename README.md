# MiniMemory AI (闪存助手) ⚡️

> 用人类验证过的“黄金事实”物理压制 AI 幻觉，提供即时纠偏、本地化、随关随走的轻量化记忆工具。

MiniMemory AI 是一个轻量级的 Chrome / Edge 浏览器扩展程序。专为重度 AI 用户（ChatGPT, Claude, Gemini, DeepSeek 等）设计。当你发现 AI 开始胡说八道，或者你在多轮对话中迷失了上下文重点时，你可以用它来**抓取事实、智能压缩、一键注入**，将 AI 强制拉回正确的逻辑轨道。


## ✨ 核心特性

- **⚡️ 极速捕捉 (Quick Capture):** 网页划选文本，按下 `Alt+X` (Mac 为 `Cmd+X`) 即可瞬间呼出面板。
- **🏷️ 多角色隔离 (Role Isolation):** 捕获的信息可打上 `[User]`, `[AI]`, `[Web]` 标签，有效防范 AI 上下文污染。
- **💉 一键注入 (Context Inject):** 勾选所需记忆，一键无缝注入当前页面的 AI 聊天框（附带强约束提示词引导）。
- **🗜️ 智能压缩 (Smart Compress):** 接入大模型 API，将冗长、充满废话的历史对话，强力浓缩为最精简的“客观事实（Atomic Facts）”，极大节省 Token。
- **📝 提示词大师 (Prompt Master):** 内置顶尖提示词生成逻辑。提取你的记忆事实，摒弃臃肿模板，生成每一个词都“承载重量 (Load-bearing)”的高密度提示词。
- **🌐 多语言自适应 & 纯文本导出:** 支持中英双语面板，支持动态保留原文语言，并提供一键导出 Markdown 文档。
- **🔌 灵活的 API 配置:** 原生支持所有兼容 OpenAI 格式的大模型 API（如 DeepSeek、通义千问、Groq、第三方中转站等）。

## 🚀 安装指南

由于目前仍在 MVP 开发阶段，你可以通过“未打包的扩展程序”模式进行本地安装：

1. **下载源码：**
   将本项目克隆或下载为 ZIP，并解压到一个固定文件夹中。
   ```bash
   git clone https://github.com/yourusername/minimemory-ai.git
   ```
2. **打开扩展程序管理页面：**
   在 Chrome 浏览器地址栏输入 `chrome://extensions/`（如果是 Edge 浏览器，请输入 `edge://extensions/`）。
3. **开启开发者模式：**
   在页面的右上角，打开**“开发者模式”**开关。
4. **加载插件：**
   点击左上角的**“加载已解压的扩展程序”**，然后选择你刚刚解压的 `minimemory-ai` 文件夹。
5. **固定在工具栏（推荐）：**
   点击浏览器右上角的“拼图”图标，将 MiniMemory AI 的闪电图标固定在工具栏上，方便随时点击打开主面板。

## 🛠️ 配置与使用

### 第一步：基础配置 (如果你需要智能功能)
1. 点击插件图标打开主面板。
2. 点击右上角的 **“⚙️ 设置”**。
3. 填入你的 **API Endpoint** (例如 DeepSeek 的 `https://api.deepseek.com/v1`，或 OpenAI 的默认地址)。
4. 填入你的 **API Key** 以及你要使用的 **Model**（如 `deepseek-v4-flash` 或 `gpt-4o`）。
5. （可选）如果你觉得默认的 `x` 快捷键不顺手，可以在这里将其修改为其他字母。
6. 点击 **“保存配置”**。

### 第二步：开始捕获记忆
1. 在任意网页（如 ChatGPT 对话页面、或者查阅的技术文档网页）。
2. **用鼠标划选**一段对你来说重要的事实、代码、或 AI 的错误回答。
3. 按下快捷键 `Alt + X` （Mac 为 `Cmd + X` 或 `Option + X`）。
4. 在弹出的悬浮按钮中，选择这段内容的来源标签（`🙋 User 提示词`、`🤖 AI 回答` 或 `🌐 网页内容`）。
5. 记忆保存成功！

### 第三步：应用记忆
打开插件面板，你会看到所有捕获的卡片。你可以：
- **一键注入上下文：** 勾选卡片，点击“⚡ 一键注入”，这些内容会被自动填入当前网页底部的聊天框中。
- **智能压缩：** 觉得卡片内容太长？全选它们，点击“🗜️ 智能压缩”，大模型会帮你提炼出字数缩减 70% 纯知识点。
- **生成 Prompt：** 勾选纯知识点，点击“📝 生成 Prompt”，大模型会为你量身定做一个带有严密逻辑结构的专业系统提示词。

## 🔒 隐私与安全性说明

- **基础功能 100% 本地运行：** 如果你不填写 API Key，划选捕捉、记忆保存、面板展示、注入与导出功能依然完全可用，所有数据仅存储在浏览器的 `chrome.storage.local` 中，**绝对不会上传至任何第三方服务器**。
- **API 通信透明：** “智能压缩”和“生成 Prompt”功能会且仅会将你**主动勾选**的卡片发送至你在设置中指定的 API Endpoint。

## 💡 开发架构
- **Vanilla 体系**：采用原生 HTML + CSS + JavaScript 编写，无冗余的重型框架。
- **Manifest V3**：遵循最新的 Chrome 插件标准，采用 Service Worker (`background.js`) 作为底层引擎。
- 采用 **Shadow DOM** 技术，确保注入网页的浮窗样式不受宿主页面 CSS 污染。
